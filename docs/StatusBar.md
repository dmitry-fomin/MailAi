## Назначение

Иконка в системной строке меню macOS со счётчиком писем и цветовой индикацией важности. Быстрый доступ к списку аккаунтов и последним непрочитанным.

## Ключевые сущности

- `StatusBarController` — владеет `NSStatusItem`, жизненный цикл привязан к `NSApplication`.
- `StatusBarIconRenderer` — отрисовка иконки с бейджем (число + цвет). Цвета:
  - **красный / акцентный** — есть важные непрочитанные (по AI-оценке);
  - **нейтральный (системный)** — есть непрочитанные, но все не важные (рассылки/уведомления);
  - **без бейджа** — все прочитано.
- `StatusBarMenu` — меню: список аккаунтов с их счётчиками, «Открыть окно…», «Настройки», «Выход».
- `AggregateCounter` — агрегирует счётчики по всем открытым аккаунтам (подписка на `MetadataStore.observe`).

## Бизнес-логика

- Иконка активна всегда, даже если ни одно окно не открыто.
- Счётчик обновляется реактивно (AsyncStream из `Storage`).
- Клик по иконке — меню; двойной клик / клик по пункту аккаунта — открывает/фокусирует окно этого аккаунта.
- При >99 непрочитанных показывается `99+` (экономим ширину).
- Анимация обновления — минимальная, без мигания.

## API

Не экспортируется — инициализируется из `AppShell` при запуске.

## Зависимости

- **От**: `Core`, `Storage`, `AppShell` (для открытия окон), AppKit.
- **Кто зависит**: —.

## Запрещено

- Рендерить в иконку содержимое писем / тему.
- Держать в памяти список всех писем — только агрегированные счётчики.
