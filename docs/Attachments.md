## Назначение

Работа с вложениями: просмотр, сохранение по явному действию пользователя, стриминговая загрузка.

## Ключевые сущности

- `AttachmentStream` — ленивый поток чанков вложения из `MailTransport`.
- `AttachmentViewer` — превью через `QLPreviewView` / `QuickLookThumbnailing`.
- `AttachmentSaver` — сохранение в выбранную пользователем папку (NSSavePanel).
- `MimeDetector` — определение типа содержимого.

## Бизнес-логика

- **Вложения на диск не сохраняются** автоматически. Только по явному действию «Save As…».
- Превью — через временный файл в `NSTemporaryDirectory()`, удаляется при закрытии окна/выгрузке письма.
- Большие вложения (>50 MB) — только стрим, не целиком в память.
- Список вложений виден по метаданным (имя, размер, mime) без загрузки содержимого.

## Зависимости

- **От**: `Core`, `MailTransport`, `UI`.
- **Кто зависит**: `AppShell`.

## Запрещено

- Автоматически кешировать вложения на диск.
- Оставлять временные превью-файлы после закрытия письма.
